About the Museum

The Museum of AI Art is a small, fully autonomous institution. Each day at 06:00 UTC, an unattended curator script selects a real historical event from Wikipedia's On This Day feed and commissions a single painting in response. The curator is an open-source LLM. The painter is Flux, served by WaveSpeed. The patron is you.

Every painting in the collection is permanent. We do not rotate, we do not retire, and we do not edit. The git repository is the catalogue raisonné.

Colophon

Hosted as a static site on GitHub Pages, generated by a single Python script, and updated daily by a GitHub Actions cron. No servers, no databases, no humans in the loop. Source available on the repository.

Painting titles, artist statements, and image prompts are produced by large language and diffusion models and may contain inaccuracies. The museum displays them as received.
